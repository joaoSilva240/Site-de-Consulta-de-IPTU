# Site de Consulta de IPTU

## Sobre o projeto
Este projeto é uma aplicação web para consulta de dados de IPTU. Ele permite que o usuário informe o número de cadastro do imóvel e visualize os dados cadastrais, áreas do terreno e construção, além do comparativo de valores e composição do imposto por exercício.

## Para que serve
A aplicação foi criada para centralizar e facilitar a consulta de informações do IPTU, exibindo:
- Dados do imóvel (endereço, áreas e cadastro).
- Valores venais por exercício.
- Composição do imposto (imposto territorial, predial, taxas e isenções).
- Observações administrativas quando disponíveis.

## Tecnologias utilizadas
- **React** com **Create React App** para a base do front-end.
- **React Router** para navegação entre páginas.
- **XLSX** para leitura de planilhas com os dados dos imóveis.
- **Lucide React** para ícones da interface.
- **CSS** para estilização do layout.
